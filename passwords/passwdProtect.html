How can Passwords be Protected?

Common ways to protect your password

There are many ways to protect your passwords. Some of them are as follows:

  • Create strong and secure passwords
  • Use a different password for each account
  • Use a safe and secure password manager to manage all your passwords (e.g. LastPass, 1Password, NordPass)
  • Use multi factor authentication
  • Never share your password
  • Be mindful of the ways that a hacker can grab your password

The benefit of creating a strong and secure password is that it is harder to guess. It is even better when it contains randomized elements.

Using the same password for multiple accounts is a rescipe for disaster, because if a hacker gets one of your passwords, the first thing they wil try is to log into your other accounts with the same password. Hence, you should always use a different password for each account.

A password manager can help store all your passwords and this can make it much easier to make long and complex passwords, because now you only have to memorize one password, the password to your password manager. Some of the best password managers are as follows:

  • Last Pass
  • 1Password
  • NordPass
  • Bitwarden
  • RememBear
  • Keeper
  • LogMeOnce

Multifactor authentication makes it so that even if a hacker obtains your password, they are now less likely to be able to get into your account because there is now another obstacle for them to cross.

Sharing your password is a terrible idea because the more people that know a secret, the more likely it is to leak.

Taking precautions to prevent hackers from obtaining your password can greatly reduce the chances of them hacking into your account.

Making safe passwords

There are many methods of making safe and secure passwords, but the key is remembering them.

The ideal password should be as long as possible and as random as possible, but how likely are you to remember something like this: Anx7S@dRsj4u-MQe. So to combat this issue, there are many techniques that allow you to create safe, secure, and randomized passwords while still making them memorable.

Correct Horse Battery Staple Method:

The Correct Horse Battery Staple method is basically choosing 4 completely random words from a dictionary and then combining them to form a password. This method allows for varying length and randomized words, and the main appeal of this method is how easy it is to remember. You can simply conjure an image that the 4 words can be used to describe making it much easier to remember the password.

Revised Passphrase Method:
This method is quite similar to the last one, but here instead of choosing random words from the dictionary, you have to come up with uncommon and bizzare words, such as Quagmire, and by coming up with multiple of this and stringing them together, you can create a difficult to guess password, such as QuagmireHancockMerciDeNada.

Sentence Method (Bruce Schneier Method):
In this method, you have to make up a sentence and transform it into a password using a rule, for example:

  • Sentence: The Old Duke is my favorite pub in South London.
  • Rule: The first two letters of every word.
  • Password: ThOlDuismyfapuinSoLo

And it is going to be quite difficult for a hacker to guess this password. Try using sentences that are personal to you, so that they are easier to remember.
